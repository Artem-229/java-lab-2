# Лабораторная работа 2

**Вариант 3:** Реализация структуры данных "Граф" и алгоритмов работы с ним

## Базовые операции графа
- Добавление/удаление вершин и рёбер
- Получение смежных вершин  
- Обход в глубину (DFS)
- Обход в ширину (BFS)

## Алгоритмы поиска кратчайших путей
- Алгоритм Флойда-Уоршелла (полная реализация)
- Алгоритм Беллмана-Форда
- Алгоритм Дейкстры (не полностью доделанная реализация)

## Графический интерфейс
- Темная тема для интерфейса
- Визуализация матрицы смежности
- Выполнение алгоритмов с подробным выводом
- Валидация ввода и обработка ошибок
